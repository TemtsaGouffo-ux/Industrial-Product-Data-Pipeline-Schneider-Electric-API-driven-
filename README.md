# Industrial-Product-Data-Pipeline-Schneider-Electric-API-driven-
Ce projet est un projet personnel visant à concevoir un pipeline data complet à partir de données produits industrielles, en s’appuyant sur une API REST documentée selon la norme OpenAPI (Swagger).  Le projet est inspiré des problématiques industrielles de gestion de catalogues produits.
